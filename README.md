# 🚀 Zomok X - Modern Social Media Platform

**Zomok X** is a high-performance, cutting-edge social media platform meticulously engineered to redefine digital interaction through a **seamless** and **intuitive** user experience. Developed as a flagship project, Zomok X leverages a sophisticated modern tech stack, primarily built on the **React.js** framework and powered by **Vite** for lightning-fast development and optimized build cycles. The application's visual identity is defined by a sleek, **aesthetic** design system utilizing **HeroUI** and **Tailwind CSS**, ensuring a fully **responsive** architecture that adapts flawlessly across all device orientations, from mobile interfaces to expansive desktop monitors.

At the core of Zomok X lies a deep integration of Artificial Intelligence. The platform features a specialized **AI Assistant** powered by the **Google Gemini API**. Unlike standard chatbots, this assistant is configured with specific **system instructions** that give it full **context** of the Zomok X ecosystem, allowing it to provide real-time, relevant support to users. This integration demonstrates advanced handling of asynchronous data streams and sophisticated prompt engineering, allowing the AI to **emerge** as a core part of the user journey rather than just a side feature.

From a structural standpoint, the project reflects professional **Systems Engineering** standards. It implements robust security protocols, including secure environment variable management to protect sensitive **credentials** like API keys, ensuring they are never **exposed** in public repositories via strict `.gitignore` configurations. The codebase is organized for **scalability**, utilizing React Hooks and the Context API for efficient state management. Furthermore, the platform integrates external data sources, showcasing the developer's ability to handle complex third-party integrations and dynamic content rendering.

Zomok X is not merely a social networking clone; it is a **validation** of modern frontend capabilities, combining high-speed performance, secure architecture, and an **interactive** AI-driven environment. Whether it is the smooth navigation of the dashboard, the real-time feedback of the streaming AI responses, or the **consistent** UI components, every aspect of Zomok X is **optimized** to provide a professional-grade social media experience, representing a significant technical **attainment**.

## 🛠️ Technical Stack
* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS & HeroUI
* **AI Engine:** Google Gemini SDK
* **State Management:** Context API & Hooks
* **Security:** Dotenv & Git-ignored credentials

## 🚀 Getting Started
1. **Clone the repo:** `git clone https://github.com/khalid-zomok/social_media_app.git`
2. **Install deps:** `npm install`
3. **Set up Environment:** Create a `.env` file and add `VITE_GEMINI_KEY=your_key`
4. **Launch:** `npm run dev`
